1. ¿Qué tan preciso fue el LLM en el primer prompt? ¿Tuviste que corregir algo manualmente?
Si era específica con lo que le decía el LLM era preciso. Pero tuve que ajustar el 
prompt ya que si  no era muy específica el LLM creaba o inventaba cosas para que 
tuviera más cuerencia. 
2. ¿Qué diferencia notaste entre un prompt vago y uno con restricciones claras?
Un prompt vago provocaba que el LLM tuviera que inventar cosas y agregarlas, por lo que
no quedaba como yo quería. Uno con restricciones permitía que el LLM no inventara 
cosas y fuera más exacto
3. ¿En qué casos preferirías escribir el código tú mismo en lugar de usar el LLM?
Al momento de iniciar y crear la estructura. Para tener claro lo que voy a hacer.
Dejar al LMM hacer lo tedioso y repetitivo
